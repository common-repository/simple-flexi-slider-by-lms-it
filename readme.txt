Simple setup responsive slider using FlexSlider by WooThemes.

== Description ==

This slider uses the FlexSlider framework by [WooThemes]. In order to display the slider the user must use the slider shortcode.

= Options that can be set: =

* Add new slider
* Add image to slider
* Slider effect
* Slideshow speed
* Animation speed


== Installation ==

1. Upload the files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the created shortcode '[lmsit_sfs id='X' /] into a post

== Frequently Asked Questions ==

= How many slider images can I add ? =

You can add up to 5 images with each slider created.
